# demo-github-ciencia-abierta

Repositorio de ejemplo creado en vivo durante la capacitación
**"GitHub como plataforma de Ciencia Abierta"** (CIEAM, Universidad Católica del Maule).

## Qué hace

`analisis.R` genera un gráfico simple de dispersión usando el dataset `mtcars`
(incluido en R), explorando la relación entre el peso de un automóvil y su
rendimiento de combustible.

## Cómo ejecutarlo

```r
source("analisis.R")
```

Esto genera `grafico_peso_vs_rendimiento.png` en la misma carpeta.
